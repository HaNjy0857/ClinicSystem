# Clinic Management System

這個專案是一個診所管理系統，用於管理病人的資料和預約。

## 功能

- **病人管理**
  - 新增、編輯和刪除病人資料
  - 查看病人列表和個別病人詳情
- **預約管理**
  - 安排、查看和取消預約
- **使用者管理**
  - 使用者登入和登出
  - 使用者身份驗證和權限管理

## 技術堆疊

- **前端**
  - 使用 EJS 模板引擎呈現頁面
  - CSS 和 Bootstrap 進行頁面樣式設計
- **後端**
  - Node.js 和 Express 框架處理後端邏輯和路由
  - 使用 Sequelize 作為 ORM 管理資料庫操作
  - Passport.js 處理使用者身份驗證
- **資料庫**
  - 使用 MySQL 作為資料庫存儲系統
